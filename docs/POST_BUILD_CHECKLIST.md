## 🐛 常见问题排查

### 编译错误
```
问题：资源文件找不到
解决：检查 drawable 文件是否存在
     检查 strings.xml 中字符串定义

问题：Kotlin 编译错误
解决：检查 MainActivity.kt 中导航逻辑
     确保移除了 R.id.nav_search 引用

问题：布局文件错误
解决：检查 navigation_menu.xml 只有 3 项
     检查所有布局文件的约束关系
```

### 运行时错误
```
问题：导航栏不显示
解决：检查 BottomNavigationView 的约束
     确保 menu 文件路径正确

问题：页面跳转崩溃
解决：检查 Intent 目标 Activity 是否在 Manifest 注册
     检查 Activity Result Launcher 是否正确初始化

问题：图标不显示
解决：检查 selector 文件引用的图标是否存在
     检查图标文件命名是否正确
```

## 📱 测试设备建议

### 最小配置
- Android 7.0 (API 24) 或更高
- 屏幕尺寸：4.5 英寸以上
- 分辨率：720p 或更高

### 推荐配置
- Android 10 (API 29) 或更高
- 屏幕尺寸：5.5-6.5 英寸
- 分辨率：1080p 或更高
- 支持生物识别

### 测试场景
1. **小屏手机**：360dp 宽度，验证布局不重叠
2. **标准手机**：411dp 宽度，验证所有元素显示正常
3. **大屏手机**：480dp+ 宽度，验证间距合理
4. **平板设备**：768dp+ 宽度，验证响应式布局
